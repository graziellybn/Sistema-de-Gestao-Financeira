# Sistema de Gestão Financeira
O Sistema de Gestão Financeira é um projeto acadêmico desenvolvido por estudantes dos cursos de Ciência da Computação e Engenharia de Software da Universidade Federal do Cariri (UFCA), como parte da disciplina de Programação Orientada a Objetos (POO). O projeto consiste em um sistema backend desenvolvido em Java, com o objetivo de permitir o gerenciamento de finanças pessoais ou compartilhadas por meio do registro, organização e análise de movimentações financeiras.

Além de atender aos requisitos da disciplina, o sistema busca aplicar, na prática, os principais conceitos da Programação Orientada a Objetos, como encapsulamento, herança, polimorfismo, abstração, associação entre classes e boas práticas de modelagem de software.

---

# 🎯​ Objetivo
O sistema busca oferecer recursos para o registro, organização e acompanhamento de receitas e despesas, permitindo ao usuário visualizar sua situação financeira por meio de consultas, relatórios e estatísticas, além de funcionalidades voltadas à análise de gastos, definição de prioridades para despesas e gerenciamento colaborativo de perfis financeiros.

---

# ⚙️​ Funcionalidades
- Cadastro e autenticação de usuários;
- Gerenciamento de receitas;
- Gerenciamento de despesas;
- Gerenciamento de categorias;
- Consulta ao histórico financeiro;
- Cálculo automático do saldo;
- Geração de relatórios financeiros;
- Geração de estatísticas financeiras;
- Classificação de despesas por grau de prioridade;
- Análise de gastos e identificação de despesas potencialmente reduzíveis;
- Compartilhamento de perfis financeiros entre usuários autorizados.
